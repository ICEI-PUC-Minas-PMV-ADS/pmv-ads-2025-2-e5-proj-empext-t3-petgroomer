# Implantação do Software

## Frontend
- Framework: Next.js (React.js)
- Linguagem: TypeScript
- Biblioteca de UI: Ant Design (antd)
- Animação: Framer Motion
- Testes: Jest, React Testing Library
- Build Tools: Babel, ESLint
- Containers: Docker (Node 18 Alpine image)
- Outros: React Icons, web-vitals

## Backend
- Framework: NestJS (Node.js, Express)
- Linguagem: TypeScript
- ORM: Prisma
- Database: PostgreSQL via Prisma
- Autenticação: JWT (JSON Web Token)
- Segurança: Helmet, cookie-parser, bcrypt
- API Docs: Swagger (via @nestjs/swagger)
- Test: Jest (unit e e2e)
- Build Tools: ESLint, Prettier
- Containers: Docker (Node 18 Alpine image)
- Deployment: Railway, Docker

## API de hospedagem de imagens (ApiFotos)
- Framework: ASP.NET Core (C#) .NET 8.0
- Hospedagem: Cloudinary (CloudinaryDotNet)
- API Docs: Swashbuckle (Swagger for .NET)
- IDE: Visual Studio Solution

## DevOps & Infrastrutura
- Version Control: Git + GitHub
- CI/CD & Hosting: Railway
- Docker
